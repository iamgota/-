# Lughat Al-Arab Language Support

This extension adds syntax highlighting and basic language support for the **Lughat Al-Arab** programming language to Visual Studio Code.

## Features

- **Syntax Highlighting** for `.عرب`, `.ه`, and `.heba` files.
- **RTL Support**: Arabic keywords and Right-to-Left writing mechanics are natively configured.

## Usage

Simply open a file with the `.عرب`, `.ه`, or `.heba` extension, and the Lughat Al-Arab language mode will be automatically applied.

## Supported Elements

- Core control structures (`اذا`, `بينما`, `لكل`)
- Object-Oriented concepts (`صنف`, `دالة`, `جديد`, `هذا`, `يرث`)
- Data types (`عدد`, `نص`, `قائمة`, `قاموس`)
- Built-in functions (`اطبع`, `اقرا`)
- Error handling (`حاول`, `التقط`, `اخيرا`, `ارمي`)
- And more!
